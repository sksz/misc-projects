HTML Tables: The Cheat Sheet

Here you can find mostly used HTML tags to create tables:

Tag Name HTML Tag Tag Description/Role
Table <table> Defines table element
Table Row <tr> Creates row in table element
Table Cell <td> Creates single cell
Table Head <thead> Defines header of table element
Table Body <tbody> Defines body of table element
Table Footer <tfoot> Defines footer of table element
